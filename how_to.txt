build/project.properties

 Bu dosya build seçenekleri için var.

	file.stylesheets = 
	extra stylesheet'lerin isimlerini buraya virgülle ayırarak yazın, bunlar en sonunda minify edilip tek dosya halinde birleştiriliyor.

	Mac OS X & Linux ortamında sisteminizde optipng ve jpegtran yüklü olması gerek, bu sayede static imajlarda yüksek oranda optimizasyon sağlıyor build script.


	file.pages = 
	
	Sunucuya yüklenecek ve içinde .css, .js linki olan dosyalar.
	
	file.exclude = 
	
	son klasörde olmasını istemediğiniz klasörler.
	

Terminal'de:

 $ cd /build
 $ ant

 # build edilen dosyalar publish klasöründe.
 # ayrıca her build sonunda (eğer yoksa sıfırdan yaratılan)
 # build/config/buildinfo.properties dosyasındaki 
 # bulid.number 1 arttırılıyor, sıfırlamak için bu build 
 # öncesinde bu dosyayı silmek yeterli.
 # CSS dosyaları kaşe edilebilmeleri için bu number ile
 # build sırasında minify edilip yeniden adlandırılıyor,
 # file.pages =  satırında belirtilen dosyalar bu yeni isimlerle
 # güncelleniyor.
